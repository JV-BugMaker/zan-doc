# 项目配置

正常配置文件都放在$ROOTPATH/resource/config/$EVN 文件加下，结构如下：

````
resource/
├── cache
├── config
│   ├── online
│   │   └── connection
│   ├── qatest
│   │   └── connection
│   ├── share
│   │   └── table
│   └── test
│       ├── connection
│       ├── kvstore
│       │   └── test
│       └── monitor
├── kvstore
└── sql
    ├── common
    └── market
````
